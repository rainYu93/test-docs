# 1 概述

# 1.1 文档概述

此文档将介绍企业微信集成SRM移动端的相关配置，主要包括企业微信配置和云SRM配置

# 2 配置说明

# 2.1 SRM配置

# 2.1.1 三方平台定义

菜单路径：移动端管理-三方平台-三方平台定义

若没有对应平台，需要新建一个三方平台，若已有对应平台，直接使用已定义的三方平台即可

[注：种子数据部署完成，此配置正常会自动生成 ]

  • 三方平台编码：wechatEnterprise
  • 描述：企业微信
  • 消息发送地址：https://qyapi.weixin.qq.com/cgi-bin/message/send
  • 获取用户地址：https://qyapi.weixin.qq.com/cgi-bin/gettoken
  • 执行代码：此代码主要是回调三方，获取用户信息，并二次握手做安全性校验

# 2.1.2 三方平台参数定义

菜单路径：移动端管理-三方平台-三方平台定义 -》 参数定义

  • REDIRECT_URL_TEMPLATE: 消息跳转地址模板（构造消息跳转链接时使用）
    https://open.weixin.qq.com/connect/oauth2/authorize?appid=APP_ID&redirect_uri=REDIRECT_URL_TEMPLATE&response_type=code&scope=snsapi_base&state=#wechat_redirect
  • JSAPI_TICKET: https://qyapi.weixin.qq.com/cgi-bin/get_jsapi_ticket?access_token=ACCESS_TOKEN 获取企业微信签名地址
  • TOKEN_URL: https://api.weixin.qq.com/cgi-bin/token 获取企业微信token地址

# 2.1.3 三方平台运营号维护

菜单路径：移动端管理-三方平台-三方平台运营号维护

# 2.1.3.1 新建一个三方运营账号

  • 账号：非必输，SRM自动生成（若需要自定义，可以维护相应的值）
  • 租户：选择企业微信对应的租户 *必输
  • 三方平台：选择步骤 2.1.1 中定义的企业微信平台（wechatEnterprise） *必输
  • 应用：根据实际使用，选择商城或者SRM *必输
  • 三方账号：企业id【见步骤：2.2.2】 *必输
  • 三方密钥：自建应用Secret【见步骤：2.2.3.2】 *必输
  • 运营号描述：租户名称+企业微信（例：甄云科技企业微信） *必输
  • 备注：若无特别需要备注的，与运营号描述相同即可
  • 平台运营账号：默认不开启（当租户为SRM平台时，可以开启，将当前三方运营号作为平台级运营账号使用）
  • 自动绑定：默认不开启（当三方账号通过接口，在srm系统中同步生成子账号时，开启该功能可实现免登录）
  • 启用：默认开启（当三方运营号不使用时，可禁用）
  • 启用客户系统认证：默认不开启（当三方系统需要使用到自己的客户系统认证账号时，可开启该功能，并结合客户系统认证执行代码使用）
  • 客户系统认证执行代码：客户系统认证流程，当启用客户系统认证标识开启时需要维护
# 2.1.3.2 获取三方运营账号生成的账号信息，用于回调地址的构造

三方运营账号信息：xxxx

# 2.1.4 三方运营参数配置


  • AGENT_ID: 企业微信自建应用AgentId（用于消息推送时，指定推送应用） *必输
  • REDIRECT_URL_TEMPLATE: 消息跳转地址模板（构造消息跳转链接时使用，优先级高于平台级配置，若不配置，则取平台级模板配置）
    https://open.weixin.qq.com/connect/oauth2/authorize?appid=xxxxxx&redirect_uri=REDIRECT_URL_TEMPLATE&response_type=code&scope=snsapi_base&state=#wechat_redirect

# 2.2.3.3 网页授权及JS-SDK配置

菜单路径：开发者接口 -》申请域名校验

配置网页授权及JS-SDK,可信域名为移动服务器域名，例如: mobile.dev.isrm.going-link.com（云开发环境）

注：默认不需要申请校验域名，若需要使用企业微信sdk，则需申请校验域名，下载验证文件放在移动服务器域名根目录下，未明确要求则不进行校验，填写域名即可

公有云项目服务器域名配置如下：

  • Dev环境：mobile.dev.isrm.going-link.com
  • Test环境：mobile.test.isrm.going-link.com
  • Pro环境：mobile.going-link.com
# 2.2.3.4 工作台应用主页设置

应用主页跳转分为srm应用和商城应用，配置分别如下：

（1）srm应用主页地址：
https://open.weixin.qq.com/connect/oauth2/authorize?appid=xxxx&redirect_uri=https%3A%2F%2Fmobile.dev.isrm.going-link.com%2FWxCover%2F&response_type=code&scope=snsapi_base&agentid=xxxx&state=wechatEnterprise-xxxx#wechat_redirect

（2）商城应用主页地址：
https://open.weixin.qq.com/connect/oauth2/authorize?appid=xxxx&redirect_uri=https%3A%2F%2Fmobile.dev.isrm.going-link.com%2FWxCover%2f%3fproxyService%3dMall&response_type=code&scope=snsapi_base&agentid=xxxx&state=wechatEnterprise-xxxx#wechat_redirect

页面地址参数说明:

  • appid：替换为企业微信的企业id【见步骤：2.2.2】
  • redirect_uri：替换为SRM方移动端跳转地址及固定入参
  • agentid：替换为企业微信自建应用AgentId【见步骤：2.2.3】
  • state：替换为wechatEnterprise-三方运营账号，【见步骤：2.1.3.2 截图中账号信息】
