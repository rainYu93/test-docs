# 组件属性

SrmCollapseItem

参数 说明 类型 是否必输 可选值 默认值 版本要求
value v-model绑定值 Boolean 否 - false v1.7.0
name 折叠面板的key Object / String / Number 是 - - v1.7.0
disabled 是否禁用 Boolean 否 - false v1.7.0
icon 右侧图标名称 String 否 - icon-chevron-down v1.7.0

SrmCollapse

参数 说明 类型 是否必输 可选值 默认值 版本要求
value v-model绑定值 Array 否 - - v1.7.0
maxOpen 最多可展开的数量 Number 否 - Infinity v1.7.0

# 组件事件

SrmCollapse

事件名称 参数 说明 版本要求
@input (val:any) 值输入改变时触发，修改v-model绑定值 v1.7.0
@change (val:any) 折叠开关状态变化时触发 v1.7.0

SrmCollapseItem

事件名称 参数 说明 版本要求
@input (val:any) 值输入改变时触发，修改v-model绑定值 v1.7.0

# 组件插槽

SrmCollapse

插槽名称 说明 版本要求
default 默认插槽,插槽子组件应该是SrmCollapseItem v1.7.0

SrmCollapseItem

插槽名称 说明 版本要求
default 默认插槽,显示默认展示区 v1.7.0
content 显示折叠区 v1.7.0
