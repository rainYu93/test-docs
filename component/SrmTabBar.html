注意

1.在单独使用srm-tab-bar组件并且应用页签个性化时（示例1第三个），如果默认页签内含有dataSet， 需要关闭掉dataSet的自动查询，改为在onIndexChange里面执行查询。 2.页签个性化code不要使用数字，应该使用英文代码。

# 在页面中使用

单独使用srm-tab-bar

    <template>
  <div>
    <srm-tab-bar :tabs="tabs"/>
    <srm-tab-bar :tabs="tabsJustify" display="justify" @reselect="reselect" @indexChange="onIndexChange"/>
    <srm-tab-bar display="justify" :tabs="tabsWithCodes" customize-code="SMBL.BUDGETMAKING.TABS" @indexChange="onIndexChange"/>
  </div>
</template>

<script>

export default {
  name: "SrmTabBarExample",
  data() {
    return {
      tabs: [ "真的这么丝滑吗", "我想试试", "？？？？？", "噢噢噢噢", "真的这么丝滑吗", "我想试试", "？？？？？", "噢噢噢噢" ],
      tabsJustify: [ "真的这么丝滑吗", "我想试试" ],
      tabsWithCodes: [
        {
          code: "underApproval",
          label: "审批中",
        },
        {
          code: "toBeSigned",
          label: "待签收",
        },
        {
          code: "feedbackUnderReview",
          label: "反馈审核",
        },
        {
          code: "toBeReleased",
          label: "待发布",
        },
        {
          code: "toBeSubmited",
          label: "待提交",
        },
        {
          code: "all",
          label: "全部",
        },
      ],
    };
  },
  methods: {
    reselect(index) {
      console.log("再次选择", index);
    },
    onIndexChange(index, oldIndex, code) {
      console.log(index, oldIndex, code);
      //在这里做切换tab刷新ds等操作
    }
  }
};
</script>

# 组件属性

SrmTabBar

参数 说明 类型 是否必输 可选值 默认值 版本要求
activeIndex 绑定当前tab下标 Number 否 - 0 --
tabs tab数组 Array<String> 否 - [] --
normalClassName 没有激活状态下tab按钮的样式 String 否 - srm-tab-bar-button-normal --
activeClassName 激活状态下tab按钮的样式 String 否 - srm-tab-bar-button-active --
onTouchClassName 按住时候tab的样式 String 否 - srm-tab-bar-button-touch --
display tab的显示方式，auto状态下tab会按元素宽度排列超出滚动。justify状态下，会按tab个数均分宽度，超出隐藏 String 否 auto/justify auto --
showTab 是否显示tab栏 Boolean 否 - true --
preload 预加载tab个数 Number 否 - 0 --
customizeCode 页签个性化编码 String 否 - null v2.1.5

SrmTabItem

参数 说明 类型 是否必输 可选值 默认值 版本要求
label 当前tab标签，组合使用时，父组件tabs属性无效 String 是 - - v1.8.0
code 页签编码，用于个性化 String 是 - - v2.1.5

# 组件事件

事件名称 参数 说明 版本要求
@reselect (index:number, oldIndex:number, code:string) 当前tab已经是active状态时重新选择tab时触发 --
@reachRight -- tab滚动到最右边触发事件 --
@reachLeft -- tab滚动到最左边触发事件 --
@scroll -- tab左右滚动事件 --
@indexChange (index:number, oldIndex:number, code:string) tab下标改变事件，index为新下标,oldIndex为就下标, code为选中的tab编码 --
