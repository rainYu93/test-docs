# 在页面中使用

点击查看代码 
<template>
  <div>
    <srm-button @click="listTreeVisible = true">点击打开数组结构弹窗</srm-button>
    <srm-tree
      height="100vh"
      :data="treeData"
      tab-key="name"
      primary-key="code"
      data-format="TREE"
      display-field="name"
      show-header
      multiple
      @select="onSelect"/>
    <srm-popup
      :visible.sync="listTreeVisible"
      height="80vh"
      :title="$T('smbl.common.message.pleaseSelect').$D('请选择')"
      rounded>
      <srm-tree
        height="80%"
        title="选择部门"
        :data="listData"
        tab-key="name"
        parent-key="parentId"
        primary-key="id"
        data-format="LIST"
        display-field="name"
        :render-fn="renderItem"
        @select="onSelect"/>
      <div class="srm-tree-foot-wrapper">
        <span class="srm-tree-foot-msg-info">
          {{
            $T("smbl.common.validation.multiple").$D("该选择为多选")
          }}
        </span>
        <srm-button fill size="large" @click="onSelect">
          {{ $T("smbl.common.validation.confirmSelect").$D("确认选择") }}
        </srm-button>
      </div>
    </srm-popup>
  </div>
</template>

# 组件属性

SrmTree 组件属性:

参数 说明 类型 是否必输 可选值 默认值 版本要求
value 选中的值 Array 否 - - v1.1.9
data 要显示的数据 Array 是 - [] v1.1.9
dataFormat 数据格式,LIST为数据结构，内部通过primaryKey&parentKey关联，TREE为树形结构数据 String 否 LIST/TREE LIST v1.1.9
renderFn 选项渲染函数，和displayField必传一个 Function 否 - null v1.1.9
loadFunction 数据加载方法，用来加载下一级数据，参数为(item, index) Function 否 - null v1.1.9
multiple 是否多选 Boolean 否 - false v1.1.9
lazy 是否懒加载 Boolean 否 - false v1.1.9
tabKey tab显示的字段名 String 是 - null v1.1.9
displayField 选项显示的字段名称，有renderFn时此属性失效 String 否 - null v1.1.9
primaryKey 数据主键 String 是 - id v1.1.9
parentKey 父级数据主键，dataFormat为LIST时必传 String 否 - pid v1.1.9
hasChildrenKey 是否有下级表示，懒加载时判断是否请求下级接口 String 否 - hasChildren v1.1.9
height 弹窗高度 String 否 - 70% v1.1.9
childrenKey 子级字段名 String 否 - children v1.1.9
showTab 是否显示导航tab Boolean 否 - true v1.1.9
showCheckbox 是否显示选择框 Boolean 否 - true v1.1.9
title 弹窗标题 String 否 - 请选择 v1.1.9
checkboxEnable 选择框是否启用，function回调参数为(item, index) Boolean/Function 否 - true v1.1.9
