介绍

个性化组件：可根据个性化配置生成页面结构，字段类型支持TEXT/INPUT/INPUT_NUMBER/SWITCH/LOV/SELECT/TEXT_AREA/EMPTY/DATE_PICKER/UPLOAD/TREE(v1.7.0) 现阶段大多数场景可以使用srm-list或者srm-form组件结合DataSet实现，不建议直接使用本组件

# 引入组件

 import {SrmCustomize} from "srm-mobile-ui";
 components: {SrmCustomize};

# 在页面中使用

点击查看代码 
    </template>
        <srm-customize :columns="getColumns" :dataSource="dataSource" code="SMBL.INQUIRY_HALL.LIST.LIST" /> 
    </template>

    <script>
    
      export default {
        data() {
          return {
            dataSource: {
              rfxTitle: "询价单12345454",
              companyName: "中国移动"
            },
          };
        }，
        computed: {
          getColumns() {
            const fileArr = [ {
              title: "附件A",
              attachmentUuid: "aaa"
            }, {
              title: "附件B",
              attachmentUuid: "bbb"
            } ];
            return [
              {
                fieldName: "询价单标题",
                fieldType: "TEXT",
                fieldCode: "rfxTitle",
                render: (val, record, item) => {
                  return <SrmButton onClick={()=>this.testClick(val, record, item)}>{val}</SrmButton>;
                },
              },
              {
                fieldName: "公司",
                fieldCode: "companyName",
                render: (val) => {
                  return <a style="color: blue" >{val}</a>;
                },
              },
              {
                fieldName: "开始时间",
                fieldCode: "approvedDate",
              },
              {
                fieldCode: "viewName",
                fieldType: "LOV",
                displayField: "viewName",
                fieldName: "询价单号",
                lovCode: "HLOD.LOV.VIEW",
              },
              {
                fieldName: "创建人",
                fieldType: "INPUT",
                inputType: "text",
                fieldCode: "createByName",
                disabled: false,
                inputChange: (val) => {
                  console.log(val.target.value, "change");
                },
              },
              {
                fieldName: "空標簽",
                fieldType: "EMPTY",
                render: () => {
                  return <SrmCalendarField theme="fill" prefix="日期" type="start-end" size="small" onInputChange={(e)=>console.log(e) }/>;
                }
              },
              {
                fieldName: "TEXT_AREA",
                fieldType: "TEXT_AREA",
                fieldCode: "reamrk",
              },
              {
                fieldName: "Switch",
                fieldType: "SWITCH",
                fieldCode: "flag",
              },
              {
                fieldName: "附件上传",
                fieldType: "UPLOAD",
                fieldCode: "UUID",
                fileArr: fileArr,
                isTab: true,
                showCount: true,
              },
              {
                fieldName: "日期",
                fieldType: "DATE_PICKER",
                fieldCode: "date",
              },
            ];
          },
        }
        
      };
    </script>

# 组件属性

参数 说明 类型 是否必输 可选值 版本要求
columns 表格列的配置描述 Array 是 - --
dataSource 数据源 Object 是 - --
code 个性化编码 String 否 - --
needPadding 是否需要外边距 Boolean 否 - --
type 来自列表页还是详情页 String 否 list, detail(默认值) --
unitType 组件类型 String 否 GRID/FORM(默认) --
isTemplate 是否开启单据样式定制 Boolean 否 - 2.1.112-alpha/2.1.37
template 单据样式模板信息，业务模块模板接口返回数据，包含字段：{templateCode, templateVersion} Object isTemplate为true时必填 - 2.1.112-alpha/2.1.37
pageCode 单据样式页面编码 String isTemplate为true时必填 - 2.1.112-alpha/2.1.37
stageCode 单据样式阶段编码 String isTemplate为true时必填 - 2.1.112-alpha/2.1.37

# 示例代码

点击查看代码 
    </template>
        <srm-customize ref="customize" :columns="columns" :dataSource="dataSource" /> 
    </template>

    <script>
    
      export default {
        data() {
          return {
            columns: [
              {
                fieldType: "INPUT_NUMBER",
                fieldName: "税务发票号码",
                fieldCode: "invoiceNumber",
                required: true,
                validator: (val) => {
                  if (String(val).length !== 8) {
                    return "长度只能为8位数字";
                  }
                  return null;
                },
              },
              {
                fieldType: "INPUT_NUMBER",
                fieldName: "校验码",
                fieldCode: "checkCode",
                required: true,
                validator: (val) => {
                  if (String(val).length !== 6) {
                    return "长度只能为6位数字";
                  }
                  return null;
                },
              }
            ],
            dataSource: {
              invoiceNumber: "",
              checkCode: "",
            },
          };
        },
        methods: {
          validate() {
            cons flag = this.$refs.customize.validate();
          }
        }
      };
    </script>

  • 标准字段无法根据配置更改类型
  • render中使用过滤器

    return this.$options.filters.calcMoney(val)
