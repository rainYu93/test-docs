# transport和dataSet方法对应关系

transportName Method
read refresh、query、loadMore
submit submit、submitSelected、submitRecords
update update、updateSelected、updateRecords
destroy remove、removeSelect、removeRecords

如果需要自定义其他transport，可以使用executeTransport方法执行请求

# dataSet事件回调

事件名称 说明 备注 版本要求
beforeQuery 执行read transport前调用
afterQuery 执行read transport后调用 支持return；return的数据作为请求结果使用；
beforeSubmit 执行submit transport前调用 支持return；return的数据作为请求参数使用；
afterSubmit 执行submit transport后调用
beforeUpdate 执行update transport前调用 支持return；return的数据作为请求参数使用；
afterUpdate 执行update transport后调用
beforeDelete 执行destroy transport前调用 支持return；return的数据作为请求参数使用；
afterDelete 执行destroy transport后调用
beforeCreate 执行dataSet.create之前调用
afterCreate 执行dataSet.create之后调用

注意：部分events回调函数支持return操作(v1.1.8及以后)，无特殊处理时，不需要return值；
