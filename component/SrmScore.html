# 在页面中使用

点击查看代码 
<template>
  <div style="height: 100%">
    五角星
    <srm-score :achieved-score="4" style-type="star" @itemClick="onItemClick"/>
    五角星+可选择一半
    <srm-score :half-flag="true" style-type="star" @itemClick="onItemClick"/>
    五角星填充+可选择一半
    <srm-score :half-flag="true" style-type="starFill"/>
    五角星填充+可选择一半+每个选择点代表的分数
    <srm-score :total-score-flag="true" :single-score-flag="true" :half-flag="true" style-type="starFill"/>
    五角星填充+可选择一半+显示总分
    <srm-score :total-score-flag="true" :half-flag="true" style-type="starFill"/>
    圆形
    <srm-score :total-score-flag="true" :margin-size="15" style-type="round"/>
    圆形+单个分数+显示总分+自定义总分
    <srm-score :total-score="100" :margin-size="15" :single-score-flag="true" :total-score-flag="true" style-type="round"/>
    五角星填充+根据已经获取的分数，渲染样式
    <srm-score :achieved-score="1.5" :total-score="5" :total-score-flag="true" :half-flag="true" style-type="starFill"/>
  </div>
</template>

<script>

export default {
  name: "SrmScoreExample",

    };
  },

  },

  methods: {
    onItemClick(currentTotalScore) {
      console.info("当前分数",currentTotalScore);
    }
  }
};
</script>

# 组件属性

SrmScore 组件属性:

参数 说明 类型 是否必输 可选值 默认值 版本要求
styleType 节点样式 String 否 round/start round v1.1.6
number 自定义节点数量 Number 否 Any 5 v1.1.6
marginSize 自定义每个打分节点间隔 Number 否 Any 32 v1.1.6
totalScore 满分分数 Number 否 Any 等于自定义节点数量，每个节点1分 v1.1.6
singleScoreFlag 节点右侧是否显示当前分数 Boolean 否 true/false false v1.1.6
totalScoreFlag 是否显示总分 Boolean 否 true/false false v1.1.6
halfFlag 是否可以打半分 Boolean 否 true/false false v1.1.6
achievedScore 已经获取的分数,根据传入值渲染组件样式 Number 否 Any 0 v1.1.6
disabled 是否禁用 Boolean 否 - false v1.1.20
