本文档记录了 srm-mobile-ui@1.1.x 中组件新增、修改、问题修复等。

# v1.1.1 ChangeLog

本节记录了 srm-mobile-ui@1.1.1 中组件新增、修改、问题修复等。

  • 发布初始版本

# v1.1.2 ChangeLog

本节记录了 srm-mobile-ui@1.1.2 中组件新增、修改、问题修复等。

  • 表单组件、个性化columns新增editor属性，用来控制组件展示形态（纯文本/编辑组件）
  • srm-list新增自动隐藏底部的插槽：bottom
  • srm-lov新增多条件筛选、支持lov配置的所有筛选字段，废弃queryField属性
  • srm-popup关闭不掉bug修复

# v1.1.3 ChangeLog

本节记录了 srm-mobile-ui@1.1.3 中组件新增、修改、问题修复等。

  • srm-list新增是否自动隐藏底部属性：autoHideBottom；加载下一页优化
  • srm-form默认保存按钮样式变更
  • 新增组件srm-tab-bar
  • srm-button新增圆形按钮属性：circle
  • srm-scroll新增高度计算策略属性：autoHeightPolicy；新增是否可以加载下一页属性：canLoadMore
  • 修复SrmCheckbox缩放动画丢失
  • 当输入框获取焦点时自动滚动到中心位置优化
  • 修复个性化代码formRow：0，被配置覆盖bug
  • 附件上传按钮优化，修复列表删除按钮显示问题

# v1.1.4 ChangeLog

本节记录了 srm-mobile-ui@1.1.4 中组件新增、修改、问题修复等。

  • 发布初始版本
  • srm-list新增支持搜索框多条件，新增显示点击效果属性：clickable
  • srm-search新增支持多条件输入
  • touch-feedback组件新增水波纹特效属性：showRipperEffect
  • srm-button、srm-action-sheet新增点击水波纹特效
  • 修复srm-switch点击无效bug
  • Customize开关组件支持inputChange
  • 输入组件新增精度控制属性：precision和多语言转换属性：localizeNumber
  • 修复srm-scroll获取不到子元素margin问题
  • srm-input-field、srm-search输入防抖优化
  • DataSet优化，dataSet删除、保存、提交相关操作统一返回Promise
  • 请求加上s-workflow-token头
  • 整体组件ui调整优化
  • lov支持已选择的显示
  • srm-popup新增标题插槽：title
  • srm-checkbox-group新增选项主键属性primaryKey
  • srm-tag-group点击优化

# v1.1.5 ChangeLog

本节记录了 srm-mobile-ui@1.1.5 中组件新增、修改、问题修复等。

  • 组件样式交互优化
  • srm-textarea-field新增是否选择字数计数器属性：showLength；修复在ios显示内边距问题
  • srm-action-sheet底部适配iPhoneX
  • srm-lov新增点击输入框字动调整高度，新增滚动列表自动隐藏键盘
  • srm-input-field修复precision为0时不处理的问题，修复让只在type=number时显示本地化字符串
  • srm-popup修复多级弹窗z-index问题
